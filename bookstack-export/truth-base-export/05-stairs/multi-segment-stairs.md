Drafting Multi-Segment Staircases (“U” Shape)
U shaped stairs are two parallel sets of straight stairs attached to a landing that creates a 180
degree turn.
Follow the points and figures below for how to draft multi-segment stairs that go both up and
down
●

Draft the center wall or railing of the “U” shaped stair. Setting the floor and using
triangulation can help.

●

Draft each segment of stairs separately.

●

Divide the stairs up from the stairs down keeping in mind to show more stairs down
than up.

●

Join the arrow segment from each stair set using a construction line (D7).

●

Delete arrow heads from all but the last stair segment.

●

Delete extra UP/DN labels and any extra gates.

●

Bind the stairs from the rest of the space using an invisible divider (D3).

Example #1: Since the center wall is a railing type wall, use a railing type wall as the division
between the stairs up and the stairs down.
