Stair Gates
Every stair section that is created will also create a stair gate, these gates are used in the virtual
tour to transport along staircases.
To link a pair of stair gates:
1. Ensure that both sets of stairs are drawn on their respective floor.
2. In E mode right-click on one of the stair gates select Pair with.
3. Choose the correct floor and gate number to pair.
4. If paired properly both gates will turn green.
○

The first number is the number of the currently selected gate, and the 2nd
number is the number of the gate it is paired with.

The number in the stair gate is used only to help identify each staircase and the number will
increase with every stair segment that is created, even if the gate or stair segment is deleted. If
a gate is paired and turns yellow, the gate is not visible to any other scans so to fix this:
1. Find the yellow stair gate.
2. Press “E” to enter edit mode
3. Left-click and drag the scan until it is visible by at least one other scan without moving it
too far away from the start of the stairs. A red line will appear projecting from each
enabled and nearby scan to verify a gate is visible.
Note: If no nearby scans are enabled, do not enable a scan so that the gate turns green. Do
not move the gate far away from the stairs so that an enabled scan (if any) can see it. In this
case, the warning can be ignored.
