Ordering and Separation of Floors
The order of the floors in the Project Explorer is determined by the photographer or operator
within another program (Stitch). The correct order would be Main Floor followed by ascending
floors and then the floors immediately descending from the Main Floor (See figure 1).
E.g. Main Floor, 2nd Floor, 3rd Floor, Lower Floor, Basement etc.

Figure 1 - Floor Order

To change the order of the floor, drag the floor name to the correct location and ensure an
arrow symbol appears.

Inspect each floor for any changes in elevation. Floors in split-level homes with the vertical
separation of greater than 4ft/1.2m or more than 6 steps of stairs are considered different
levels, so the floors should be separated. Homes with the vertical separation of less than
4ft/1.2m or less than 6 steps are considered one level and should be combined.

Figure 3 - Split Vs. Combined

An exception to this rule above is based on how the photographer submitted the file. If the
photographer submitted the file with all the floors separate even though there’s only a couple
steps between floors we will leave it as submitted.
Another exception to this rule is for Landings, Foyers or Foyers in typical ranch style homes
where stairs up or down are the next immediate area. With these cases, drafters should
combine the spaces unless there’s significant overlap between floors.

Backsplits
Backsplit homes have very specific rules so that we can properly report both interior area
totals and exterior area totals.
1. In back split homes where the floors have not been combined, the stairs leading up
must be bound with a type 3 line for proper calculation of the exterior perimeter. All
stairs leading up in a backsplit home will be labelled “Stairs up
(duplicated/backsplit)” except for the lowest floor.
2. All stairs leading down in a backsplit home will be included.
