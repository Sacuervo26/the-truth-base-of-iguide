Site Plans – Drafting & Detailing
Before Drafting a Site Plan
Please make sure the following checklist is completed before you start drafting any Site Plan:
✓ Respective Floor Plan is Fully Finished
o The main iGUIDE Floor Plan has been finalized.
o Exterior scans were properly aligned and used to obtain the thickness of the
perimeter wall.
o Remember that all exterior areas, except for Balconies, are now going to be drafted
directly in the Site Plan.
✓ Nearmap Aerial Image Obtained
o The aerial image of the property should have already been exported from
Nearmap.
o The Lot Area was obtained and typed in Draft.
o Please refer to the “Site Plans – Nearmap” document for the full explanation on
this process.

Additional Rules – Building Layout
Main Building and any other secondary buildings in the property (Detached Garage, Sheds, etc)
must be represented using standard D1 walls (18 cm/7 in) that are aligned to the exterior edge
of the construction.
As explained in the previous section, the D1 boundary will be automatically created in the SP floor
when selecting “Create Site Plan”. These walls, regardless of their thickness in the floor plan, will
all be re-sized to 7in/18cm. Furthermore, all walls will already be aligned to the exterior edge of
the property to ensure the perimeter dimensions remain the same.
Warning:
The automatic D1 walls, generated with the “Create Site Plan” option, may not work as intended
in the following two situations:
1. The D1 walls surrounding Bay Windows and Ledges (areas that are
Outdoor/Protrusion/Duplicated) will not be replicated

Figure # 4 – Copying Outdoor/Protrusion/Duplicated Features

2. Very thick D1 walls, when replicated in the SP floor, may not properly be aligned to the
exterior perimeter edge as intended.

Roof Geometry
Use the Nearmap image as a reference to trace all the ridges/valley lines of the roof using D7
lines.
•

Do not account for roof overhangs between 1'-0" to 1'-6". The roof outline must be drawn
starting precisely at the interior edges and corners of the D1 outline.

•

Please remember the Site Plan is only meant to provide spatial context of the property;
it is not supposed to be 100% accurate. Therefore, prioritize that the roof layout is as
visually appealing as possible.

Enclosing & Labeling Outdoor Areas
The next step is to limit and label the different exterior spaces that can be represented.
Patio, Porch, & Deck labels are still going to be used in the Site Plan floor. Additionally, Site Plans
will have 5 more Room Types & Labels that can be used:
•

GARDEN:
o Use for defined planting areas, flower beds, vegetable patches, or shrubbery
zones that are distinct from the lawn.

•

GRASS:
o Use for open lawn areas. "Walkable" surfaces that are not used to grow plants
(trees, bushes, flowers, etc).

•

POOL:
o Use for swimming pools (both in-ground and above-ground), hot tubs, or
significant water features.

11

o Any in-ground water features such as ponds or fountains should be enclosed as
their own "room", given the "POOL" room type and then manually change the
label as POND or FOUNTAIN accordingly.
•

DRIVEWAY:
o Use for paved or gravel areas specifically designed for vehicle access and parking.

•

WALKWAY:
o Use for paved paths, or sidewalks intended for pedestrian navigation around the
property.

Walls/lines in the Site Plan should be placed to be as close as possible to the visual information:
•

Visible Dividers (D6): Used to divide every space within the Site Plan that does not have
a railing/half wall/full height wall.

•

Railings (D5): In the Site Plan floors, D5 walls will be used to represent fences or any other
railing-like feature. Drafters are allowed to add doors on top, if needed, to match the
visual information.

•

Structures (D7): As mentioned before, D7 lines will be used to outline the roof geometry.
Additionally, this line type will be utilized to show steps, stairs and any other floor detail
that needs to be shown.

•

Half Walls (D4): In case the lot is enclosed by half walls instead of a fence, D4 walls can
be used. Stand-alone half walls can also be shown if needed.

•

Exterior (D1) & Interior (D2): D1 walls (7in/18cm) are used for building outlines. Wall
protrusions that are adjacent to a building can also be shown with a full-height wall

•

Every other boundary needed to limit the spaces mentioned above, needs to be drafted
using D6 lines.

Warning: Exterior stairs & steps do not need to be enclosed as separate spaces. Additionally, stairs
that have more than three steps need to have an arrow and a manual label.

Objects & Gates
As mentioned before, the Site Plan Object category has over 40 outdoor options (trees, fire pits,
patio furniture, among others). Once all the spaces have been properly divided, place objects to
match the satellite imagery and the visual data.
•

Objects should be scaled realistically to provide a proper understanding of space.
Please avoid oversized and undersized elements.

Warning: Every Site Plan, regardless of whether the task is Standard or Premium, should have
objects placed.
Finally, every Site Plan should be connected to the respective Floor Plan using Stair Gates.
•

Gates need to be placed at every human sized entry/exit point. By this definition,
Overhang Garage doors would not have connecting gates.

•

Gates in the Floor Plan should be placed inside the property and beside the
corresponding exit point.

•

Gates in the Site Plan should be placed outside of the property’s outline and beside
the corresponding exit point.
