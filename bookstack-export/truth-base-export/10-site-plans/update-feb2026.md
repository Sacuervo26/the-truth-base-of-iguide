Site Plans – Update (February 10th, 2026)
Site Plan Setup
Previous Method
To create a Site Plan (SP), drafters would need to right-click the property address and select “Add Site Plan”.
This would create a Property Plans building and a Site Plan floor. Later in the process, a D6 boundary
would be created by tracing the exterior face of the building walls from the Main Floor.

Updated Method
The process of initializing the Site Plan has been streamlined to automate the footprint creation. Instead
of selecting “Add Site Plan”, right-click the Main Floor and select “Create Site Plan”.

Figure # 1 – Create Site Plan Option

This action will automatically generate the Property Plans and the Site Plan floor. Nevertheless, this time
a D1 boundary will automatically appear in the Site Plan floor replicating the layout of the Main Floor.

Building Layout
Previous Method
Every building in a SP would be drawn using a D6 layout traced to the exterior face of the building walls.
This would require the drafter to copy and paste the floor plan walls into the SP floor, trace the D6 lines
and finally delete all original walls and additional features from the floor plan.

Updated Method
From now on, the Main Building and any other secondary buildings in the property (Detached Garage,
Sheds, etc) must be represented using standard D1 walls (18 cm/7 in) that are aligned to the exterior
edge of the construction.
As explained in the previous section, the D1 boundary will be automatically created in the SP floor when
selecting “Create Site Plan”. These walls, regardless of their thickness in the floor plan, will all be resized to 7 in/18 cm. Furthermore, all walls will already be aligned to the exterior edge of the property to
ensure the perimeter dimensions remain the same.

Warning:
The automatic D1 walls, generated with the “Create Site Plan” option, may not work as intended in the
following two situations:
1. The D1 walls surrounding Bay Windows and Ledges (areas that are
Outdoor/Protrusion/Duplicated) will not be replicated

2. Very thick D1 walls, when replicated in the SP floor, may not properly be aligned to the exterior

These two scenarios must be manually corrected by the drafter. Rely on exterior laser data to properly

Main Building Room Type:
Previous Method
Drafters would label the building area as “COMMERCIAL” and then manually rename it as “MAIN
BUILDING”.

Updated Method
Draft v2.19.11 or above have a new label called “MAIN BUILDING”. From now on, please give this label
to the building area.
“MAIN BUILDING” has as ROOF room type.

Shed, etc), please label as “MAIN BUILDING” and then manually change to the appropriate label.
