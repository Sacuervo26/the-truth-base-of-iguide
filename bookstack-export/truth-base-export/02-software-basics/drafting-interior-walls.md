Drafting Interior Walls
Drafting of interior walls is the bulk of the drafting process.
●

Draft interior walls of the property by using line type 2 (D2).

●

Draft walls that originate at exterior wall vertices first.

●

Ensure that they are snapped to the exterior wall.

●

Draft interior walls using wall thickness and door jams as references.

Press “E” to enter edit mode.

Highlight wall segment (select multiple segments using ctrl + left click).
