Drafting Software
The following is an overview of the Project Explorer, Toolbar, Pano Preview and the Drafting
Environment.

Project Explorer (A)
The Project Explorer allows for fast navigation through the structure of the project. A property
is a container for multiple buildings. Each project is a container for multiple buildings (i),
multiple floors (ii) and multiple scans (iii). Each floor contains exactly one floor plan with
multiple scans.

Figure 2 - Project Explorer

2

Scans:
The following are options when right clicking a scan: Rename; Enable/Disable in Portal; Delete;
Fix Laser Data; Set as initial Pano; Change Color. Selected scans are highlighted while others
are faded.

Figure 3 - Scan Options Context Menu

Buildings, floors and scans can be reordered using drag & drop. The order is usually defined in
Stitch and used on the Portal (in the floor selector menu and slideshow). This order should not
be changed unless formally requested. Middle click to enable/disable the visual data of scans
and floors in Draft quickly.

Pano Preview (C)
This feature shows the full 360 panorama for the selected scan. A projection of the point cloud
data is shown around the middle of the image to help the user figure out where the laser is
registering.

Figure 5 - Pano Preview

The camera height and ceiling height properties can be adjusted by middle-clicking positions
below or above the center of the image. The Mouse wheel is used to scroll up and down to
view the entire pano. Double-click the image to create shaded radar lines with points of
interest (POI) that can be used to help align the point cloud data. Right-click to remove these
lines.

Figure 6 - Shaded Radar Lines

4

In Drawing mode (D mode) the Pano Preview can be used for direct drawing if the camera
height is properly set (useful for railings and countertops). Another possibility is to draw using
triangulation. If two scans are selected (see S Mode) mark POIs in the first panorama and then
double-click the same POIs (in the same order) in the second panorama to intersect the radar
lines. The current POI is indicated with a red dot.

Drafting Environment (D)
This is the drafting area where the laser data will appear when the file is opened. This is where
the Drafter will complete the bulk of the drafting process including, laser data alignment,
drafting walls, labeling rooms, placing doors and windows.
