Wall/Line Types
Below is a description of each line/wall type used in Draft.
Cosmetic Wall/Line: When cosmetic walls overlap it will NOT create a common point/node at
the intersection.
Structural Wall/Line: When structural walls overlap it will create a common point/node at the
intersection.

Exterior Wall Type
D1 (Structural): This wall type is used for the outside (exterior) perimeter walls of
the property/home and the size or thickness of the line is dictated by what the
operator or photographer enters in Stitch.

Interior Wall Type
D2 (Structural): This wall type is used for the floor to ceiling (full height) inside
(interior) walls of the property/home. The thinner this wall gets the more yellow it
becomes. The thicker this wall gets the more blue it becomes.

Invisible Divider Line Type
D3 (Structural): This wall type is used to separate spaces of the home that have
no physical walls to divide the rooms/spaces of the property/home.

Half Wall Type
D4 (Structural): This wall type is used for the half walls (from the floor up to
about waist level) of the property. This wall will always remain white no matter
how thick or thin it is.

Railing Wall Type
D5 (Structural): This wall type is used for the railings inside of the
property/home. This wall type typically shouldn’t need to be thickened or
thinned. This wall type will always remain white and should never be used to
draft any exterior space.

Visible Divider Line Type
D6 (Structural): Used to outline built outdoor structures adjoining or abutting
the subject building (e.g. decks, balconies, patios, car-ports, porches), not yards or
other landscaping (enclosed by fencing or otherwise). Also used to outline interior
ledges.

Structures Line Type
D7 (Structural): This line type is used to draw stairs, to show 1 step between
sunken floors and for indicating fireplaces.

Object Line Type
D8 (Cosmetic): This line type is used only for premium drafts. This includes
built-in countertops, built-in cupboards, built-in cabinetry, refrigerators, undercounter dishwashers, sinks, bathtubs, showers, laundry washers and dryers, etc.
Premium objects will be added in for kitchens, bars, bathrooms and laundry
rooms (excluding furniture).

Cosmetic Wall Type
D9 (Cosmetic): This wall type is used to draw a physical wall inside of a
bathroom that separates the sink/shower area from a closed off toilet room.
This makes the wall invisible to the area calculation of the room allowing the
space to have 1 label instead of 2.
